# OWEM - Open Wiki of Electrochemical Methods

This repository contains the source code and workflows that are required to serve the [OWEM website](https://owem.github.io).

Edits to the wiki should be made in its repository: [anderfuller/owem](https://github.com/anderfuller/owem).

The website is built using the [Just the Docs](https://just-the-docs.github.io/just-the-docs/) template for [Jekyll](https://jekyllrb.com/)
